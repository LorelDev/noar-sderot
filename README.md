# נוער שדרות

פלטפורמה לנוער: פותחים נושא, מצביעים, מגיבים ומתייעצים עם יועץ AI — ודשבורד ניהול ליחידת הנוער.

**אתר חי:** https://noar-sderot.web.app
**דשבורד מנהלים:** https://noar-sderot.web.app/dashboard.html

## ארכיטקטורה

| קובץ | תפקיד |
|---|---|
| `index.html` + `app.js` | האתר לנוער: נושאים, הצבעות, תגובות, היועץ |
| `dashboard.html` + `dashboard.js` | דשבורד מנהלים: מדדים, נושאים מובילים, ניהול סטטוסים |
| `store.js` | שכבת נתונים מול Firebase (Auth + Firestore, עדכונים חיים) |
| `firebase-config.js` | הגדרות: פרטי הפרויקט, רשימת מנהלים, מפתח AI |
| `firestore.rules` | חוקי אבטחה — נפרסים עם `firebase deploy` |

## תשתית (פרויקט Firebase: `noar-sderot`)

- **Authentication** — כניסה עם Google, סשן נשמר (כניסה אוטומטית בביקור הבא).
- **Firestore** (אזור me-west1, תל אביב) — נושאים, תגובות והתייעצויות בזמן אמת.
- **Hosting** — האתר עצמו.

### חוקי האבטחה בקצרה

- קריאה פתוחה לכולם; כתיבה רק למחוברים.
- נושא חדש חייב להיפתח עם האימייל של הכותב.
- נוער יכול לעדכן רק הצבעות ומונים; שינוי סטטוס ומחיקה — מנהלים בלבד.
- רשימת המנהלים מופיעה **פעמיים**: ב-`firebase-config.js` (תצוגה) וב-`firestore.rules` (אכיפה). כל שינוי — לעדכן בשניהם ולהריץ `firebase deploy --only firestore`.

## היועץ (צ'אט בוט)

צ'אט משותף פר נושא: כל ההתייעצויות של כולם נשמרות, והיועץ עונה עם ההקשר המלא — התיאור, ההצבעות, כל התגובות וכל מה שנשאל לפני כן.

- בלי מפתח API: מצב בסיסי (סיכום הדיון + המלצות מעשיות).
- עם `anthropicApiKey` ב-`firebase-config.js`: Claude אמיתי. **אזהרה:** מפתח בדפדפן גלוי לכל מבקר — מתאים לפיילוט קצר בלבד; לייצור מעבירים את הקריאה ל-Cloud Function.

## פיתוח ופריסה

```bash
# הרצה מקומית
python3 -m http.server 4173

# פריסה (אתר + חוקים)
firebase deploy --project noar-sderot --only hosting,firestore
```
